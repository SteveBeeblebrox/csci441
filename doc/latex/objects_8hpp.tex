\doxysubsection{Detailed Description}
Helper functions to draw 3D Open\+GL 3.\+0+ objects. 

\begin{DoxyAuthor}{Author}
Dr. Jeffrey Paone
\end{DoxyAuthor}
\begin{DoxyCopyright}{Copyright}
MIT License Copyright (c) 2017 Dr. Jeffrey Paone
\end{DoxyCopyright}
These functions draw solid (or wireframe) 3D closed Open\+GL objects. All objects are constructed using triangles that have normals and texture coordinates properly set.

NOTE\+: This header file will only work with Open\+GL 3.\+0+
