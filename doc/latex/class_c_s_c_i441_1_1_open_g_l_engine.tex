\textbackslash{}desc Used to setup everything GLFW related. This includes the Open\+GL context and our window.~\newline
~\newline
 By default, completes the following steps in order\+:~\newline

\begin{DoxyItemize}
\item sets error callback~\newline

\item initializes GLFW~\newline

\item sets window hints for Open\+GL MAJOR.\+minor~\newline

\item requests Core Profile~\newline

\item requests double buffering~\newline

\item marks window as resizable or not based on constructor creation~\newline

\item creates a window~\newline

\item makes the window the current context~\newline

\item sets the swap interval to 1 \begin{DoxyNote}{Note}
This method should be overridden if any additional callbacks need to be registered. When registering additional callbacks, the parent implementation must be called first and then the additional callbacks may be registered as desired. 
\end{DoxyNote}
