README

CSCI441 OpenGL Library GitHub Repo

This library is intended to be used with OpenGL for CSCI441/444/544 at the Colorado School of Mines.

When building, the library must be compiled and linked against:

  • OpenGL
  • GLFW
  • GLEW
  • glm
